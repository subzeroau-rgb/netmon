API reference

GET /api/switches — list all devices
POST /api/switches — add a device (type: aruba | mikrotik)
PUT /api/switches/:id — update a device
DELETE /api/switches/:id — remove a device
POST /api/switches/:id/toggle — enable / disable
POST /api/switches/:id/test — test credentials / SNMP
GET /api/live?switch=id — latest interface samples
GET /api/history?switch=id&interface=&minutes=
GET /api/summary?switch=id&period=hour|day|week
GET /api/overview — aggregate per device
GET /api/topology — topology graph + live link traffic
PUT /api/topology/nodes — save node positions
POST /api/topology/links — add a link
DELETE /api/topology/links/:id — remove a link
